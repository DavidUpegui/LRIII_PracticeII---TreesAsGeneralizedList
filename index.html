Instrucciones de uso

1. Ingresar el árbol

Para ingresar el árbol como hilera siga las siguientes reglas:

  • El árbol siempre debe iniciar con paréntesis abierto
  • Todo paréntesis abierto debe cerrarse
  • Use las letras (solo una letra) para indicar un elemento del árbol
  • Use los paréntesis abiertos para asignar los hijos del elemento anterior, por ejemplo el árbol (a(b,c)) se refiere a un elemento a con hijos b y c.
  • Use las comas para separar los hermanos, por ejemplo en el arbol (a(b,c)) los elementos b y c son hermanos.
  • Use los paréntesis cerrados para delimitar los hijos de uno de los elementos, por ejemplo en el árbol (a(b(c,d),f)) , c y d son son hijos de b.
  • El primer elemento en un árbol es la raíz, la raíz no tiene hermanos. Por ejemplo, el arbol (a,b) o el arbol (a(b,c),d) está mal escrito.
  • Nota: El árbol más pequeño no vació es (a), un árbol con s único elemento la raíz.

El árbol deberá ingresarse en la barra de texto. El botón para ingresar el árbol se activará únicamente cuando algún valor haya sido ingresado. Una vez haya ingresado el árbol presione el botón ingresar.

En el caso en que haya ingresado el árbol erróneamente le saltará un mensaje advirtiéndoles cuál pudo haber sido su equivocación.

Si el árbol fue ingresado satisfactoriamente se mostrará una pantalla en la que podrá encontrar la siguiente información:

  • El árbol que acaba de ingresar.
  • La altura del árbol.
  • El grado del árbol.
  • El número de hojas del árbol.
  • Una barra de texto para buscar un elemento del árbol.
